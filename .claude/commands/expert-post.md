이 세션은 **기술/학문 글**을 쓴다. 전문가와 입문자를 함께 아우르되, 아이들 교육용 글은 다른 세션(`/edu-post`)이 담당하므로 여기서는 쓰지 않는다.

작성 전 반드시 확인:
- `_meta/PERSONA.md` — 특히 §10(이미지·도식), §11(깊이·구체성: 제품·수치·실사례 필수, 비유는 실제 설명 뒤에만, 글 유형별 깊이 정책)
- `_meta/CATEGORIES.md`, `_meta/SERIES_OUTLINE.md`, `_meta/IDEAS.md` 큐 우선순위
- `_meta/NOTES.md` — 현재 상태·깊이 보강 백로그·다음 할 일

깊이 정책(PERSONA §11):
- **AX·하드웨어·기술 심화글**: 깊이 있는 설명 + 실사례(실제 제품·시스템·수치) + 도식(mermaid) 필수. 기준선 = `ax-06-npu`.
- **개념 정리(기초)글**: 쉽게 시작 → 실제가 복잡해도 끝까지. "쉬우니 얕게" 금지.
- 하드웨어는 반드시 제품군 예시(국산 포함)와 스펙. 수치는 웹검색으로 검증.

운영: 세션 중에는 cron 기다리지 말고 수동 발행(`python scripts/publish_one.py --slug <슬러그>` → commit/push). 작업 후 NOTES 갱신.
